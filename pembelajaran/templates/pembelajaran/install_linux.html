Distribusi Linux & Fokus Utama: Ubuntu

Ubuntu

Filosofi: “I am because we are.” — Filosofi Ubuntu tentang kebersamaan.

Dibuat oleh: Canonical Ltd (2004). Turunan dari Debian, berfokus pada kemudahan & dukungan jangka panjang.

Desktop: GNOME.

sudo apt update && sudo apt upgrade

Fakta: Nama rilis Ubuntu selalu hewan + sifat, misal “Jammy Jellyfish”.

Debian

Diluncurkan: 1993 — fondasi banyak distro termasuk Ubuntu.

Filosofi: Kebebasan software dan stabilitas tinggi.

sudo apt-get update
cat /etc/debian_version

Fakta: Nama “Debian” dari “Debra” dan “Ian” — pasangan pendirinya

Fedora

Dibuat oleh: Red Hat (2003). Playground fitur baru sebelum ke RHEL.

Desktop: GNOME terbaru, cepat adopsi teknologi baru.

sudo dnf update
sudo dnf install neofetch

Fakta: “Fedora loves freedom, friends, features, and first.”

Arch Linux

Motto: Keep It Simple, Stupid (KISS).

Ciri khas: Rolling release, pengguna membangun sistem dari nol.

sudo pacman -Syu
neofetch

Fakta: Pengguna Arch sering bilang: “btw, I use Arch”

openSUSE

Dibuat oleh: Komunitas Jerman, didukung SUSE Linux GmbH.

Versi: Leap (stabil) & Tumbleweed (rolling).

sudo zypper refresh
sudo zypper update

Fakta: YaST (Yet another Setup Tool) adalah tool admin GUI legendaris!

Linux Mint

Tujuan: Pengalaman desktop halus & ramah pengguna.

Berbasis: Ubuntu, DE utama: Cinnamon, MATE, XFCE.

sudo apt install mintupdate

Fakta: Logo daun melambangkan kesegaran — Ubuntu versi manis

Manjaro

Turunan: Arch Linux dengan tampilan lebih ramah.

Kelebihan: Update cepat tanpa ribet setup manual.

sudo pacman -Syu
mhwd-kernel -li

Fakta: Dirilis tahun 2011 dan cepat populer di kalangan gamer Linux

Pop!_OS

Dibuat oleh: System76, fokus pada produktivitas dan hardware modern.

Desktop: COSMIC berbasis GNOME, desain clean & efisien.

sudo apt install pop-desktop
gnome-control-center

Fakta: Salah satu distro favorit pengguna laptop dan kreator konten

Distro Linux Mana yang Cocok untuk Kamu?

Fokus: Ubuntu

Ubuntu adalah distro paling populer untuk desktop Linux. Dirancang agar mudah digunakan dan memiliki dukungan jangka panjang.

  • Model rilis: reguler (6 bulan) dan LTS (2 tahun).
  • Ubuntu 24.04 LTS (Noble Numbat) didukung hingga 2029.
  • Flavors: Kubuntu, Xubuntu, Lubuntu, Ubuntu MATE, Budgie, Kylin, Studio, dan lainnya.

APT & Paket Manajemen di Ubuntu

APT adalah manajer paket utama di Ubuntu, didukung backend dpkg. Gunakan perintah berikut:

sudo apt update && sudo apt upgrade
sudo apt install package-name
sudo apt autoremove

Selain APT, Ubuntu juga mendukung Snap dan Flatpak untuk aplikasi modern yang terisolasi.

GNU Tools Umum di Semua Distro

  • grep — mencari teks dalam file.
  • sed — manipulasi teks otomatis.
  • awk — analisis data baris demi baris.
  • lsblk — melihat disk & partisi.
  • top / htop — monitoring sistem.
  • uname -a — melihat kernel & arsitektur.
  • tar — mengarsipkan dan mengekstrak file.
  • chmod & chown — mengatur izin file.

Panduan Lengkap Menginstall Linux

Terdapat berbagai metode untuk menginstal Linux di perangkatmu.
Pilih pendekatan yang paling sesuai dengan kebutuhan dan tingkat kenyamananmu.

1. Dual Boot (Windows + Linux)

  • Unduh file ISO distro Linux pilihanmu dari situs resminya (misalnya Ubuntu, Mint, Fedora).
  • Buat USB bootable menggunakan Rufus atau balenaEtcher.
  • Masuk ke BIOS dan atur urutan boot agar USB berada di posisi pertama.
  • Jalankan instalasi Linux dan pilih opsi Install alongside Windows untuk membuat dual boot.

2. Virtual Machine

  • Install software virtualisasi seperti VirtualBox atau VMware.
  • Buat mesin virtual baru, pilih file ISO Linux sebagai sumber boot.
  • Jalankan mesin virtual dan eksplor Linux tanpa mengubah sistem utama.

Cocok untuk latihan, pengujian, dan belajar tanpa risiko.

3. WSL (Windows Subsystem for Linux)

  • Buka PowerShell (Run as Administrator), lalu jalankan:
wsl --install
  • Pilih distro seperti Ubuntu dari Microsoft Store.
  • Buka melalui Terminal Windows untuk pengalaman Linux langsung di Windows.

4. Live USB

  • Boot dari USB berisi Linux tanpa melakukan instalasi permanen.
  • Berguna untuk troubleshooting atau mencoba Linux sebelum instalasi penuh.

Perubahan tidak akan tersimpan setelah restart.

Tips: Pastikan semua data penting telah dicadangkan sebelum memodifikasi partisi atau sistem operasi.
